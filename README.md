# next-hacker-news
A Hacker News clone (PWA) with offline functionality built with Next.js

âœ… Production: https://next-hacker-pwa.vercel.app/

### Install dependencies

```bash
npm install
# or
yarn
```

### Build

#### Static export

```bash
npm run export
# or
yarn export
```

To serve it yourself, you can run:

```bash
npx serve -s out
```

#### Server hosted

```bash
npm run build
# or
yarn build
```

To serve it yourself, run:

```bash
npm start
# or
yarn start
```
